В этом репозитории представлено простое веб-приложение на Flask, которое взаимодействует с базой данных PostgreSQL, развернутой с использованием Docker Compose. 
Приложение было добавлено в GitLab, который также развернут как контейнер с помощью Docker Compose. В качестве инструмента непрерывной интеграции (CI) использовался TeamCity.
